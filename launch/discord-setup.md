# Discord Server Setup

## Server Name
**Axint Community** (or "Agentic Empire" if we want the umbrella brand)

## Channel Structure

### Information
- **#welcome** — Auto-post on join. Rules, links, getting started
- **#announcements** — Release notes, breaking changes, events (admin only)
- **#roadmap** — Link to GitHub roadmap, voting on priorities

### Community
- **#general** — Freeform discussion
- **#showcase** — Share what you've built with Axint
- **#help** — Get help with compilation issues, setup, etc.
- **#ideas** — Feature requests and brainstorming

### Development
- **#contributing** — For people working on Axint itself
- **#beta** — Pre-release builds, testing coordination
- **#wwdc** — Apple API tracking, WWDC reactions

### Roles
- **@Core Team** — Nima + contributors with merge access
- **@Beta Tester** — Early access program members
- **@Contributor** — Anyone who's had a PR merged

## Welcome Message

```
Welcome to the Axint community! 👋

Axint is the open-source compiler that turns TypeScript defineIntent() calls into native Apple App Intents AND MCP tool servers.

**Quick links:**
• GitHub: https://github.com/agenticempire/axint
• Playground: https://axint.ai/#playground
• Docs: https://docs.axint.ai
• Install: `npm install -g @axintai/compiler`

**Getting started:**
1. Try the playground at axint.ai
2. Install with npm and run `axint init`
3. Check out #help if you get stuck
4. Share what you build in #showcase!

**Rules:**
• Be kind and constructive
• No spam or self-promotion (unless it's Axint-related)
• Use threads for long discussions
• Bug reports go on GitHub Issues, not here
```

## Bot Setup
- **GitHub bot**: Post new releases, PRs, and issues to #announcements
- **Welcome bot**: Auto-assign @Member role, send DM with getting-started guide
