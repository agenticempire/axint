# Agentic Empire

We build open-source tools that give AI agents native superpowers on every platform.

### Flagship Project

**[Axint](https://github.com/agenticempire/axint)** — The open-source compiler that transforms TypeScript agent definitions into native Apple App Intents. TypeScript in, Swift out.

```bash
npm install -g axint
```

### What We Believe

AI agents shouldn't be trapped inside chat windows. They should live natively in Siri, Shortcuts, Spotlight, and every surface your users already touch. Axint makes that possible with a single `defineIntent()` call.
